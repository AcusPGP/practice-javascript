## 1. Initialization
- Initialize a string by using double quotes or single quotes
```javascript
     // Declare variables

     var fullName_1 = "Hello 'World'";
     var fullName_2 = 'Hello \'World\'';

     console.log(fullName_1); // Hello 'World'
     console.log(fullName_2); // Hello 'World'
```
