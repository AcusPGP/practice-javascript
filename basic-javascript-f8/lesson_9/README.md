# Lesson 9: Loops

## for
```js
     for (expression 1; expression 2; expression 3) {
          execution code;
     }
```
## for/in
```js
     for (variable in array) {
          execution code;
     }
```
## for/of
```js
     for (variable of array) {
          execution code;
     }
```
## while
```js
     while (condition) {
          execution code;
     }
```
## do/while
```js
     do {
          execution code;
     } while (condition)
```
